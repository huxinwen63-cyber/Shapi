# Numi Beta Materials / Numi 测试招募材料

Copy-ready, bilingual (EN + 中文) materials for recruiting and running the Numi beta test.
Replace every **[bracketed]** placeholder before use.

可直接使用的中英双语材料，用于招募和开展 Numi beta 测试。使用前请替换所有 **[方括号]** 占位内容。

## Contents / 目录

| File | What's inside / 内容 |
|------|----------------------|
| `01-outreach-letters.md` | Outreach letters with a clear **ask** for: parents, teachers, schools, learning specialists, dyscalculia organizations, pediatric learning clinics. / 面向家长、老师、学校、学习专家、计算障碍机构、儿童学习诊所的招募信，每封含明确诉求。 |
| `02-flyers-and-announcement.md` | Parent recruitment flyer, school/community flyer, and digital announcement (short + long). / 家长招募传单、学校/社区传单、线上公告（长短两版）。 |
| `03-consent-info-guide.md` | Parent consent form, beta info sheet, family participation guide, and app walkthrough document. / 家长同意书、测试信息说明、家庭参与指南、应用操作演示文档。 |
| `04-emails-and-google-form.md` | Follow-up, reminder, and thank-you email templates, plus the Google Form question list. / 跟进、提醒、感谢邮件模板，以及 Google 表单题目清单。 |

## Sign-up channels / 报名渠道
- **In-app form:** Home → "Join the Beta" (collects name, email, child's age, role, device, referral, consent). / **应用内表单：** 首页 →"加入 Beta 测试"。
- **Google Form:** Build from the question list in `04-emails-and-google-form.md`. / **Google 表单：** 依据 `04` 文件中的题目清单搭建。

## Suggested timeline / 建议流程
1. Recruit via letters, flyers, and announcement → collect sign-ups. / 通过招募信、传单、公告招募 → 收集报名。
2. Send follow-up + consent form to interested families. / 向有意向家庭发送跟进邮件与同意书。
3. Share access + participation guide + walkthrough. / 发送使用权限、参与指南与操作演示。
4. Mid-beta reminder email. / 测试中期发送提醒邮件。
5. Collect feedback (survey). / 收集反馈（问卷）。
6. Send thank-you email. / 发送感谢邮件。

> These are plain-language templates, not legal advice. Review the consent form and data practices
> against the privacy laws that apply to you before collecting data from children.
> 以上为通俗语言模板，不构成法律意见。收集儿童数据前，请依据适用的隐私法规审核同意书与数据处理方式。
